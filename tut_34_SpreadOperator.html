Spread Operator - feature of ES6

for e.g we have an two arrays and we want include both that time we can used spread operator as follows, 

        let arr1 = [1, 2, 3];

        let arr2 = [4, 5, 6];
    
        let arr3 = [...arr1, ...arr2];
    
        console.log(arr3);
If we have add after 5 the and before 6, 
        let arr4 = [4, 5, ...arr1, 6]
        console.log(arr4);
