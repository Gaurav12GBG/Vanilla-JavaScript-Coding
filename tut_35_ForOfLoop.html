for...of Loop in JS - Features from ES6

In this features, It is not needed to specify the particular string or array length for iterating the loop. basically, In normal array it is needed.

        let scores = [80, 95, 65, 48];

        for (const x of score) {
            console.log(x);
        }
We can used it for string also, 

        let str = ["gaurav", "raghav", "amar", 'harry'];

        for(let x of str){
            console.log(x);
        }
