Template Literals - Features from ES6

Template Literal does not work in double/single quote, It is work in only backtick(``).

        Basically we are writing the string as follows,
        
        let str = "Hello World";
        console.log(str);

        if we have to write both the words in new line then we are using \n for that as follows,

        let str = "Hello\nWorld";
        console.log(str);

        We are creating the string using backtick and which is also called as template literals.
        We can write whatever format we want as like that,

        e.g 01:

        let str = `Hello
                   World`;

        Output should be,

        Hello
        World

        e.g 02:

        Normally we can used like,
        str = "gaurav";
        console.log("My name is" + str + ".");

        Using template Literals,

        str = "gaurav";
        console.log(`my name is ${str}. Thank you.`);
